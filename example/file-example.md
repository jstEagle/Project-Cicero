### $i^{something}$
$$i^2 = -1$$
$$i^3 = i^2*i = -1i = -i$$
$$i^4 = i^2*i^2 = 1$$
$$i^5 = i^2*i^2*i = i$$
$$\dots$$
$$i^{4n} = 1$$
$$i^{4n+1} = i$$
$$i^{4n + 2} = -1$$
$$i^{4n+3} = -i$$
